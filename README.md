# Cigar Order Hub

B2B SaaS central ordering hub for cigar retailers and wholesalers.

## Quickstart (for devs)

**Backend:**
cd backend
npm install
npm start

Runs on http://localhost:4000

**Frontend:**
cd frontend
npm install
npm start dev

Runs on http://localhost:3000

## MVP Features Planned
- Retailer/supplier accounts with license storage
- Access approvals
- Multi-supplier ordering
- QuickBooks + UPS/USPS integrations
